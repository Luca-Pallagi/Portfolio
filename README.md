# Luca Pallagi Porfolio

## Projektbeschreibung
Dies ist mein persönliches Portfolio, erstellt, 
um meine Fähigkeiten in Webentwicklung, React und UI/UX Design zu präsentieren.
Es zeigt meine Projekte, Fähigkeiten und bietet eine interaktive, 
moderne Benutzeroberfläche mit subtilen metallischen Effekten und 
eleganten Hover-Animationen.

## Technologie
React – Komponentengesteuerte Benutzeroberfläche
CSS – Für Styling, Animationen und Metallic-Design
React Social Icons – Integration von sozialen Links
Animationen & Effekte – Subtile metallische Shine-Animationen, Glanz-Effekte und Hover-Interaktionen
